## Deploy en GitHub Pages
1. Crea un repositorio nuevo (p. ej., `qr-tiendas-pwa`).
2. Sube todos los archivos de esta carpeta a la **raíz** del repo (index.html, app.js, sw.js, manifest, icons).
3. En GitHub: *Settings → Pages* → **Deploy from Branch** → `branch: main`, `folder: /(root)`.
4. Abre la URL que te da Pages (ej.: `https://<usuario>.github.io/qr-tiendas-pwa/`).

Requiere HTTPS para usar la cámara.
